About/Help
npFontUnify by @trippygeese

npFontUnify is a simple web based application to unify Nepali ASCII Font faces in a docx or a txt files.

With npFontUnify you can easily convert your documents typed in Preeti, Sagarmatha, PCS Nepali or Kantipur to Devanagari Unicode or Preeti. Simply start by Uploading your document by clicking the upload button. If your document is supported, you will get to choose the Map from and Map to list. You can choose Autodetect which will find the all supported fonts within the document and convert them to Unicode/Preeti while keeping non Nepali fonts as it is or you can force an origin font by which all the text in document will be taken as selected font and will be mapped to Unicode

How to use

1) Click on Upload icon
2) Browse your file and upload it
3.a) If your file is ".txt" select the font mapping which should be used (From font -> To font)
3.b) If your file is ".docx" select the font mapping which should be used or better use the "Auto" Option to auto convert supported font while keeping other font intact
4) If your file is ".docx" select the components which should be processed
5) Wait ! If any errors report
6) Click download icon

Note: This tool can convert text on paragraphs, text boxes/shapes and tables only as of now, headers/footers or everything else may be ignored.

If you encounter any problem or have any suggestions open an issue on github. (Github link in footer )
